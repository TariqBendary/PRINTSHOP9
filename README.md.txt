# مشروع برنت شوب - Print Shop

موقع ويب لطلب خدمات الطباعة أونلاين مع حساب الأسعار، وحساب تكلفة التوصيل بناءً على العنوان، وإرسال الطلب عبر واتساب، مع توليد QR Code لرابط الواتساب ليسهل مشاركته أو حفظه.

---

## مكونات المشروع

يتكون المشروع من الملفات التالية:

- `index.html` → الصفحة الرئيسية تحتوي على النموذج.
- `style.css` → تنسيقات وتصميم الموقع.
- `script.js` → الأكواد البرمجية لتنفيذ العمليات الحسابية، وإرسال الطلب، وتوليد QR Code.
- `README.md` → هذا الملف يشرح المشروع.

---

## طريقة الاستخدام

1. افتح الملف `index.html` في المتصفح.
2. املأ بيانات الطلب:
   - نوع الطباعة
   - عدد النسخ
   - عدد الأوراق
   - نوع الورق
   - عنوان التوصيل
   - الملاحظات الإضافية (إن وجدت)
   - رفع الملفات (اختياري)
3. سيظهر لك حساب الأسعار تلقائياً.
4. اضغط على زر **إرسال عبر الواتساب**.
5. سيتم فتح واتساب مع رسالة الطلب تلقائياً.
6. سيظهر أسفل النموذج QR Code يمكنك مسحه للوصول لنفس رابط الواتساب.

---

## ميزات المشروع

✅ يدعم لغتين (عربي / إنجليزي).  
✅ حساب سعر الطباعة تلقائياً.  
✅ حساب تكلفة التوصيل بناءً على المسافة.  
✅ توليد QR Code لرابط الواتساب.  
✅ تصميم متجاوب وجميل.
